Tools to Teach These Topics

Topic Area Primary Teaching Tools & Platforms Why These Tools
AI Orientation PowerPoint/Google Slides, Whiteboard/Jamboard, VS Code + Markdown Explain fundamentals, compare AI/ML/DL, showcase model families and ethics checklists.
Prompt-Engineering ChatGPT / Claude / Gemini, Promptfoo, VS Code REST Client/Thunder Client, Jupyter/Colab Live prompt demos, A/B testing of prompts, show message roles & parameters.
LLM Ecosystem (open-source) ollama, Text-Generation-WebUI (opt.), vLLM, Docker Run local models (Mistral/Llama), demonstrate latency, context windows & safety.
Embeddings (concepts & coding) Python, sentence-transformers, Hugging Face Hub, NumPy/Pandas, Matplotlib Generate vectors, compute cosine similarity, visualize, batch & normalize.
Vector DB — Chroma ChromaDB (chromadb), FastAPI, SQLite/Postgres Quick local store; teach collections, metadata filtering, CRUD.
Vector DB — FAISS FAISS (CPU/GPU), SciPy, Annoy/HNSWlib Index types (IVF, HNSW), recall vs speed trade-offs, PQ/OPQ.
Managed Vector DBs Pinecone, Weaviate Cloud, Qdrant Cloud Hands-on SaaS: namespaces, replication, filtering, monitoring dashboards.
DB Management (indexes, partitions, upserts) Chroma/FAISS/Pinecone SDKs, Postman/Insomnia, Grafana/Prometheus (opt.) Create & monitor indexes, partitions, upserts, measure latency/throughput.
Semantic Search workflow Python notebooks, ragas (eval), spaCy for NER, rank-bm25, FastAPI Full pipeline: preprocessing → embedding → upsert → query → rerank → evaluate.
NER & NLP utilities spaCy, Hugging Face Transformers, Presidio (PII redaction) Extract entities, clean text, demonstrate helper functions and privacy.
RAG demo (end-to-end) LangChain / LlamaIndex, FastAPI/Flask, Streamlit/Gradio, Chroma/FAISS/Pinecone Combine embeddings + vector DB + LLM; minimal QA app with evaluation.
Security & Responsible AI Model Cards, OWASP LLM Top 10, Azure/OpenAI safety docs, prompt-injection tests Teach risks, red-teaming, guardrail design patterns.
Data Fundamentals & Prep Pandas, PyArrow, DuckDB, Apache Parquet, unstructured/pymupdf loaders Chunking, token counting, metadata schema, efficient file formats.
Experiment Tracking MLflow, Weights & Biases Track runs, parameters, models, retrieval metrics.
Deployment Docker & Compose, Render/Fly.io/Heroku, or Azure/AWS/GCP Containerize vector service + API + UI for reproducible demos.
Tip: Pair each section with a 10–15 minute live demo notebook and a small assignment (e.g., build a tiny RAG over 10 docs with Chroma).
